## 目录结构

小程序的页面目录结构非常清晰，全部放在pages下面

```shell
├─ pages  
│  ├─ activity              # 活动主页面
│  ├─ activity-comment      # 活动评论页面
│  ├─ activity-custom       # 发布自定义活动页面
│  ├─ activity-declare      # 活动积分申报页面
│  ├─ activity-detail       # 活动详情页面
│  ├─ activity-entry        # 活动成员审核页面
│  ├─ activity-release      # 活动发布页面
│  ├─ activity-score        # 活动积分页面
│  ├─ activity-search       # 活动查询页面
│  ├─ activity-sideLight    # 活动发布花絮页面（未上线）
│  ├─ activity-template     # 活动模版发布页面
│  ├─ group                 # 群组主页面
│  ├─ group-custom          # 发布自定义群组页面
│  ├─ group-detail          # 群组详情页面
│  ├─ group-job             # （没啥用）
│  ├─ group-search          # 活动查询页面
│  ├─ index                 # 精选首页
│  ├─ integralDetail        # 用户积分详情页面
│  ├─ log                   # （没啥用）
│  ├─ my                    # 我的首页
│  └─ news                  # 新闻的webview页面

```
