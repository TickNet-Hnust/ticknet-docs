这里介绍目录结构

## src

`src`的目录结构

```shell
├─ assets             # 静态资源
├─ componenets        # 开发者封装的组件
├─ element-ui         # elementUI注册相关，不用管
├─ element-ui-theme   # elementUI主题相关，不用管
├─ icons              # 字体图标相关
├─ mock               # mockAPI，不用管
├─ router             # Vue路由
├─ store              # Vue状态管理
├─ utils              # js工具函数
├─ views              # 视图
├─ App.vue            # 入口视图
└─ main.js            # 入口
```
